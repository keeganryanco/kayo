# KAYO

KAYO is a monorepo containing:

- `apps/ios`: SwiftUI iOS app (`iOS 17+`) managed by XcodeGen
- `apps/web`: Next.js landing site + minimal API surface for Vercel
- `packages/brand`: shared brand tokens consumed by both app surfaces

## Quick start

```bash
pnpm install
pnpm run generate:assets
pnpm run build
pnpm run test
```

### iOS

```bash
cd apps/ios
xcodegen generate
xcodebuild -scheme KAYO -destination 'platform=iOS Simulator,name=iPhone 16' build
```

### Web

```bash
pnpm --filter @kayo/web dev
```

## Workflow rules

- RFC required for major changes (architecture, core features, security/data collection changes)
- Keep files small and focused; avoid monolithic modules
- Generated artifacts should be generated by scripts, not edited manually

See [`docs/CONTRIBUTING.md`](docs/CONTRIBUTING.md) and [`docs/rfcs/`](docs/rfcs/) for process details.
